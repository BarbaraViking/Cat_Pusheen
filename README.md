## 📝 Описание

Данный проект позволяет **поухаживать** за виртуальным котиком ***Пушином**. В проекте есть 6 категорий активностей: 
   **- Рисовать**  
   **- Есть** (можно будет поесть различную еду)  
   **- Играть** (в проекте вас ждет 3 мини-игры)  
   **- Читать** (мини-история про пушина)  
   **- Спать**  
   **- Гулять**  
   И самое крутое:  
   **- Pusheen AI** (общение с моделью 4o-mini)  

---

## 🛠 Технологии

- **C#** — язык приложения.  
- **Windows Forms** — платформа для создания графического интерфейса.    
- **4o-mini** — Большая языковая модель для диалога с пользователем.  
- Для хранения **токена** для языковой модели использовалась библиотека **DotNetEnv** (файл .env)  

---
## 🚀 Установка и запуск

Для локального запуска потребуется клонировать репозиторий и прописать зависимости.  

1. git clone https://github.com/BarbaraViking/Cat_Pusheen.git  
2. cd "Cat Pusheen"  
3. dotnet restore  
4. dotnet build
